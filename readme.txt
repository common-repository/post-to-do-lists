== Description ==

Create to-do lists on your posts and pages similar to those used in Basecamp and Backpack.

= Features =
* Add multiple lists to a post, page or custom post type
* Add and sort list items
* Add a date to list items with the jQuery UI Datepicker
* Check items off as done
* Output the list in your content using a custom shortcode

== Installation ==

1. Upload the 'post-to-do-lists' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start adding to-do lists to your posts, pages, and custom post types
